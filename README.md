# Advent of Code
This repository contains my solutions to advent of code puzzles, for each year I participated. 

## What is Advent of Code
Per the creator: "Advent of Code is an Advent calendar of small programming puzzles for a variety of skill levels that can be solved in any programming language you like. People use them as interview prep, company training, university coursework, practice problems, a speed contest, or to challenge each other."

For more information on advent of code see (the official website)[https://adventofcode.com/2025/about]
